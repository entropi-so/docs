# Quickstart

## Prerequisites

- Flutter SDK matching Dart `^3.10.1`.
- Android Studio or Xcode for device builds.
- Python 3.10 for the backend.
- PostgreSQL with pgvector, or the provided Docker Compose database.

## 1. Install mobile dependencies

```bash
cd mobile-palme
flutter pub get
```

## 2. Configure backend environment

Create a `.env` file in `core-palme` with at least:

```bash
DATABASE_URL=postgresql://atrex:atrex04@localhost:5432/entropi_mvp
SECRET_KEY=replace-with-a-long-random-secret
APP_ENV=development
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
```

## 3. Start backend services

With Docker Compose:

```bash
cd core-palme
docker compose up --build
```

Or run FastAPI directly after installing requirements:

```bash
cd core-palme
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

## 4. Run mobile app

Android emulator:

```bash
cd mobile-palme
flutter run -d emulator-5554
```

iOS simulator or physical device:

```bash
flutter run -d <DEVICE_ID> --dart-define=API_BASE_URL=http://localhost:8000
```

Physical iOS release devices should use an HTTPS backend URL:

```bash
flutter run --release -d <DEVICE_ID> --dart-define=API_BASE_URL=https://core-palme-production.up.railway.app
```
