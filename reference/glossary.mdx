# Glossary

| Term | Meaning |
| --- | --- |
| Palme | A saved product memory. Often represented by a `Product`. |
| Universe | User-facing name for a board/list. |
| Board/List | Backend `lists` table row. |
| Product | Saved product/capture/import row. |
| Capture | Camera or gallery image sent to the vision pipeline. |
| Stumped | Product status used when capture analysis failed but a row was saved. |
| Ready | Product status for completed captures/imports. |
| Purchased | Product status after user marks it bought. |
| Friend permission | A `list_permissions` row granting visibility or add capability. |
| Share token | Public token used by shared board links. |
| Ask | AI assistant UI backed by `/api/palme-advisor/chat`. |
| Product marker | Assistant text marker like `(ID 18)` parsed into a product preview card. |
