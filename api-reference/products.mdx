## Endpoint

```http
GET /products/{product_id}
```

## Overview

Returns a single product row by ID. Since capture is synchronous,
[`POST /pipeline/analyze`](/api-reference/analyze) already returns the
finished product — this endpoint is a plain fetch by id (for detail views,
refetches, or reading a product created by another flow).

## Request

### Authentication

```http
Authorization: Bearer <access_token>
```

### Path parameters

<ParamField path="product_id" type="integer" required>
  The product ID.
</ParamField>

### cURL

<CodeGroup>
```bash cURL
curl -X GET "http://localhost:8000/products/42" \
  -H "Authorization: Bearer $TOKEN"
```

```python Python
import requests

resp = requests.get(
    f"http://localhost:8000/products/{product_id}",
    headers={"Authorization": f"Bearer {TOKEN}"},
)
resp.raise_for_status()
product = resp.json()
```

```javascript JavaScript
const res = await fetch(`http://localhost:8000/products/${productId}`, {
  headers: { Authorization: `Bearer ${token}` },
});
const product = await res.json();
```
</CodeGroup>

  "merchant": "Sézane",
  "brand_confidence": 0.9,
  "raw_text": "SÉZANE",
  "price": 145.0,
  "currency": "EUR",
  "product_url": null,
  "image_url": "data:image/jpeg;base64,/9j/...",
  "error_message": null,
  "board_ids": [1],
  "created_at": "2026-07-04T02:39:26.359572-07:00"

### A stumped capture

```json
{
  "id": 43,
  "status": "stumped",
  "name": null,
  "image_url": "data:image/jpeg;base64,/9j/...",
  "error_message": "Capture failed: Visual understanding returned no data.",
  "board_ids": [1],
  "created_at": "2026-07-04T02:39:26.359572-07:00"
}
```

## Fields

<ParamField body="id" type="integer">
  Product ID.
</ParamField>

<ParamField body="status" type="string">
  `ready` or `stumped` (or `purchased` once bought). The deprecated
  `peeking`/`framing`/`sleuthing` values are never emitted.
</ParamField>

<ParamField body="content_type" type="string | null">
  Coarse lifestyle bucket: `place`, `clothing`, `decor`, `food`, or `other`.
</ParamField>

<ParamField body="name" type="string | null">
  Short natural name of the subject (e.g. `forest-green wool jacket`).
</ParamField>

<ParamField body="description" type="string | null">
  Rich, natural description of what is visible. The primary driver of search
  quality.
</ParamField>

<ParamField body="category" type="string | null">
  Main category (e.g. `outerwear`, `cafe`, `lamp`, `pasta dish`).
</ParamField>

<ParamField body="object_type" type="string | null">
  Specific object type (e.g. `wool jacket`, `armchair`, `burger`).
</ParamField>

<ParamField body="tags" type="string | null">
  Comma-separated style tags, colors, and setting folded together.
</ParamField>

<ParamField body="merchant" type="string | null">
  Brand — only when it is actually readable in the image, above the
  confidence threshold; otherwise `null`. Never guessed from style.
</ParamField>

<ParamField body="brand_confidence" type="float | null">
  Confidence attached to `merchant` (0–1). `null` when no brand was read.
</ParamField>

<ParamField body="raw_text" type="string | null">
  Verbatim text read in the image.
</ParamField>

<ParamField body="price" type="float | null">
  Price, only when visibly present in the image.
</ParamField>

<ParamField body="currency" type="string | null">
  Currency code (`EUR`, `USD`, ...). Defaults to `EUR`.
</ParamField>

<ParamField body="product_url" type="string | null">
  Always `null` — capture involves no web lookup.
</ParamField>

<ParamField body="image_url" type="string | null">
  Base64 data-URI. The cropped subject on `ready`; the raw photo on
  `stumped`.
</ParamField>

<ParamField body="error_message" type="string | null">
  Populated when `status = stumped`. `null` otherwise.
</ParamField>

<ParamField body="board_ids" type="integer[]">
  IDs of the lists/boards this product belongs to.
</ParamField>

<ParamField body="created_at" type="timestamp">
  Row creation timestamp (ISO 8601 with timezone).
</ParamField>

## Error responses

### 401 Unauthorized

Missing or invalid bearer token.

### 404 Not Found

No product with that ID exists, or it is not owned by the caller (existence
is not leaked).

```json
{ "detail": "Product not found" }
```
